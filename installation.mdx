## Requirements

- Go 1.24 or later

## Install the SDK

There are two ways to add the SDK to your project.

<CodeGroup>

```bash go get
go get -u github.com/clerk/clerk-sdk-go/v2
```

```go import (Go modules)
import "github.com/clerk/clerk-sdk-go/v2"
```

</CodeGroup>

<Note>
If you add the import to your Go file and run `go mod tidy`, the Go toolchain will fetch the module automatically. You don't need to run `go get` separately.
</Note>

## Module path

The module path for v2 is:

```
github.com/clerk/clerk-sdk-go/v2
```

All resource packages are sub-paths of this module. Import only the packages for the resources you need.

## Import specific resource packages

The SDK is organized into resource packages that mirror the [Clerk Backend API](https://clerk.com/docs/reference/backend-api). Import each package individually.

```go
import (
    "github.com/clerk/clerk-sdk-go/v2"              // core types and SetKey
    "github.com/clerk/clerk-sdk-go/v2/user"          // Users API
    "github.com/clerk/clerk-sdk-go/v2/organization"  // Organizations API
    clerkhttp "github.com/clerk/clerk-sdk-go/v2/http" // HTTP middleware
)
```

Common packages and what they provide:

| Package | Purpose |
|---|---|
| `github.com/clerk/clerk-sdk-go/v2` | Core types, `SetKey`, `SetBackend`, helpers |
| `github.com/clerk/clerk-sdk-go/v2/user` | Users API |
| `github.com/clerk/clerk-sdk-go/v2/organization` | Organizations API |
| `github.com/clerk/clerk-sdk-go/v2/http` | HTTP middleware (`WithHeaderAuthorization`, `RequireHeaderAuthorization`) |
| `github.com/clerk/clerk-sdk-go/v2/jwt` | JWT decoding and verification |
| `github.com/clerk/clerk-sdk-go/v2/jwks` | JSON Web Key Set client |

## Configure the API key

After importing the core package, call `clerk.SetKey` once at startup with your Clerk secret key.

```go
import "github.com/clerk/clerk-sdk-go/v2"

clerk.SetKey("sk_live_XXX")
```

Your secret key is available in the [Clerk Dashboard](https://dashboard.clerk.com) under **API Keys**.

<Warning>
Do not commit your secret key to source control. Read it from an environment variable instead:

```go
import (
    "os"
    "github.com/clerk/clerk-sdk-go/v2"
)

clerk.SetKey(os.Getenv("CLERK_SECRET_KEY"))
```
</Warning>
