## Get started in four steps

<Steps>
  <Step title="Install the SDK">
    Add the SDK to your Go module:

    ```bash
    go get github.com/clerk/clerk-sdk-go/v2
    ```
  </Step>
  <Step title="Set your secret key">
    Configure the SDK with your Clerk secret key. Find it in the [Clerk Dashboard](https://dashboard.clerk.com) under **API Keys**.

    ```go
    import "github.com/clerk/clerk-sdk-go/v2"

    clerk.SetKey("sk_live_XXXX")
    ```
  </Step>
  <Step title="Call an API">
    Import the package for the resource you need and call the operation:

    ```go
    import (
        "context"
        "github.com/clerk/clerk-sdk-go/v2/user"
    )

    users, err := user.List(context.Background(), &user.ListParams{})
    ```
  </Step>
  <Step title="Add authentication middleware">
    Protect your HTTP handlers with Clerk's built-in middleware:

    ```go
    import clerkhttp "github.com/clerk/clerk-sdk-go/v2/http"

    mux.Handle("/protected", clerkhttp.RequireHeaderAuthorization()(handler))
    ```
  </Step>
</Steps>
