```go
import "github.com/clerk/clerk-sdk-go/v2/session"
```

## The Session object

<ResponseField name="ID" type="string">
  The unique identifier for the session.
</ResponseField>

<ResponseField name="Object" type="string">
  Always `"session"`.
</ResponseField>

<ResponseField name="ClientID" type="string">
  The ID of the Clerk client that owns this session.
</ResponseField>

<ResponseField name="UserID" type="string">
  The ID of the user this session belongs to.
</ResponseField>

<ResponseField name="Status" type="string">
  Current session status. Possible values: `"active"`, `"revoked"`, `"ended"`, `"expired"`, `"removed"`, `"abandoned"`, `"replaced"`.
</ResponseField>

<ResponseField name="LastActiveOrganizationID" type="string">
  The ID of the organization the user was last active in, if any.
</ResponseField>

<ResponseField name="LatestActivity" type="*SessionActivity">
  Information about the most recent activity for this session.

  <Expandable title="SessionActivity fields">
    <ResponseField name="ID" type="string">
      The unique identifier for the activity record.
    </ResponseField>
    <ResponseField name="Object" type="string">
      Always `"session_activity"`.
    </ResponseField>
    <ResponseField name="DeviceType" type="*string">
      The device type, e.g. `"desktop"` or `"mobile"`.
    </ResponseField>
    <ResponseField name="IsMobile" type="bool">
      Whether the session originated from a mobile device.
    </ResponseField>
    <ResponseField name="BrowserName" type="*string">
      Name of the browser used, if known.
    </ResponseField>
    <ResponseField name="BrowserVersion" type="*string">
      Version of the browser used, if known.
    </ResponseField>
    <ResponseField name="IPAddress" type="*string">
      IP address of the client, if available.
    </ResponseField>
    <ResponseField name="City" type="*string">
      City derived from the IP address, if available.
    </ResponseField>
    <ResponseField name="Country" type="*string">
      Country derived from the IP address, if available.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="Actor" type="json.RawMessage">
  If this session was created via impersonation, contains the actor's claims.
</ResponseField>

<ResponseField name="LastActiveAt" type="int64">
  Unix timestamp (milliseconds) of the session's last activity.
</ResponseField>

<ResponseField name="ExpireAt" type="int64">
  Unix timestamp (milliseconds) when the session expires.
</ResponseField>

<ResponseField name="AbandonAt" type="int64">
  Unix timestamp (milliseconds) when the session is abandoned.
</ResponseField>

<ResponseField name="CreatedAt" type="int64">
  Unix timestamp (milliseconds) when the session was created.
</ResponseField>

<ResponseField name="UpdatedAt" type="int64">
  Unix timestamp (milliseconds) when the session was last updated.
</ResponseField>

## Get

```go
func Get(ctx context.Context, id string) (*clerk.Session, error)
// Client method:
func (c *Client) Get(ctx context.Context, id string) (*clerk.Session, error)
```

<CodeGroup>
```go Package-level
s, err := session.Get(context.Background(), "sess_abc123")
```

```go Client struct

cfg.Key = clerk.String("sk_live_...")
```

</CodeGroup>

## List

Returns a paginated list of sessions.

```go
func List(ctx context.Context, params *ListParams) (*clerk.SessionList, error)
// Client method:
func (c *Client) List(ctx context.Context, params *ListParams) (*clerk.SessionList, error)
```

`clerk.SessionList` has two fields: `Sessions []*clerk.Session` and `TotalCount int64`.

### ListParams

<ParamField path="Limit" type="*int64">
  Maximum number of results to return.
</ParamField>

<ParamField path="Offset" type="*int64">
  Number of results to skip.
</ParamField>

<ParamField path="ClientID" type="*string">
  Filter sessions belonging to a specific Clerk client ID.
</ParamField>

<ParamField path="UserID" type="*string">
  Filter sessions belonging to a specific user.
</ParamField>

<ParamField path="Status" type="*string">
  Filter by session status. Possible values: `"active"`, `"revoked"`, `"ended"`, `"expired"`, `"removed"`, `"abandoned"`, `"replaced"`.
</ParamField>

### Example

<CodeGroup>
```go Package-level
list, err := session.List(context.Background(), &session.ListParams{
    ListParams: clerk.ListParams{Limit: clerk.Int64(20)},
    UserID:     clerk.String("user_abc123"),
    Status:     clerk.String("active"),
})
```

```go Client struct
list, err := c.List(context.Background(), &session.ListParams{
    ListParams: clerk.ListParams{Limit: clerk.Int64(20)},
    UserID:     clerk.String("user_abc123"),
    Status:     clerk.String("active"),
})
```
</CodeGroup>

## CreateToken

Creates a session token for the specified session. If a JWT template name is provided, the token is built using that template.

```go
func CreateToken(ctx context.Context, params *CreateTokenParams) (*clerk.SessionToken, error)
// Client method:
func (c *Client) CreateToken(ctx context.Context, params *CreateTokenParams) (*clerk.SessionToken, error)
```

`clerk.SessionToken` has two fields: `Object string` and `JWT string`.

### CreateTokenParams

<ParamField path="ID" type="string" required>
  The session ID.
</ParamField>

<ParamField path="TemplateName" type="string">
  The name of a JWT template to use when generating the token. Leave empty to use the default template.
</ParamField>

<ParamField path="ExpiresInSeconds" type="*int64">
  Custom expiry for the token in seconds.
</ParamField>

### Example

<CodeGroup>
```go Package-level
token, err := session.CreateToken(context.Background(), &session.CreateTokenParams{
    ID:           "sess_abc123",
    TemplateName: "my-template",
})
```

```go Client struct
token, err := c.CreateToken(context.Background(), &session.CreateTokenParams{
    ID:               "sess_abc123",
    ExpiresInSeconds: clerk.Int64(3600),
})
```
</CodeGroup>
