## Create

Creates a new user.

```go
func Create(ctx context.Context, params *CreateParams) (*clerk.User, error)
// Client method:
func (c *Client) Create(ctx context.Context, params *CreateParams) (*clerk.User, error)
```

### CreateParams

<ParamField path="EmailAddresses" type="*[]string">
  Email addresses to associate with the user.
</ParamField>

<ParamField path="PhoneNumbers" type="*[]string">
  Phone numbers to associate with the user.
</ParamField>

<ParamField path="Web3Wallets" type="*[]string">
  Web3 wallet addresses to associate with the user.
</ParamField>

<ParamField path="Username" type="*string">
  The user's username.
</ParamField>

<ParamField path="Password" type="*string">
  The user's plaintext password.
</ParamField>

<ParamField path="FirstName" type="*string">
  The user's first name.
</ParamField>

<ParamField path="LastName" type="*string">
  The user's last name.
</ParamField>

<ParamField path="ExternalID" type="*string">

</ParamField>

<ParamField path="PublicMetadata" type="*json.RawMessage">

<ParamField path="PrivateMetadata" type="*json.RawMessage">
  Private metadata for the user.

<ParamField path="UnsafeMetadata" type="*json.RawMessage">
  Unsafe metadata for the user.

## List

Returns a paginated list of users. Internally makes two API calls: one to fetch the users and one to fetch the total count.

```go
func List(ctx context.Context, params *ListParams) (*clerk.UserList, error)
// Client method:
func (c *Client) List(ctx context.Context, params *ListParams) (*clerk.UserList, error)
```

`clerk.UserList` has two fields: `Users []*clerk.User` and `TotalCount int64`.

### ListParams

<ParamField path="Limit" type="*int64">
  Maximum number of results to return.
</ParamField>

<ParamField path="Offset" type="*int64">
  Number of results to skip.
</ParamField>

<ParamField path="OrderBy" type="*string">
  Field to sort by. Prefix with `-` for descending order.
</ParamField>

<ParamField path="Query" type="*string">
  Free-text search across name, email, phone number, and username.
</ParamField>

<ParamField path="EmailAddressQuery" type="*string">
  Filter by email address (partial match).
</ParamField>

<ParamField path="PhoneNumberQuery" type="*string">
  Filter by phone number (partial match).
</ParamField>

<ParamField path="UsernameQuery" type="*string">
  Filter by username (partial match).
</ParamField>

<ParamField path="NameQuery" type="*string">
  Filter by name (partial match).
</ParamField>

<ParamField path="Banned" type="*bool">
  Filter to only banned (`true`) or non-banned (`false`) users.
</ParamField>

<ParamField path="EmailAddresses" type="[]string">
  Filter by exact email address match. Accepts multiple values.
</ParamField>

<ParamField path="ExternalIDs" type="[]string">
  Filter by external ID. Accepts multiple values.
</ParamField>

<ParamField path="PhoneNumbers" type="[]string">
  Filter by exact phone number match. Accepts multiple values.
</ParamField>

<ParamField path="Web3Wallets" type="[]string">
  Filter by Web3 wallet address. Accepts multiple values.
</ParamField>

<ParamField path="Usernames" type="[]string">
  Filter by exact username match. Accepts multiple values.
</ParamField>

<ParamField path="UserIDs" type="[]string">
  Filter by user ID. Accepts multiple values.
</ParamField>

<ParamField path="OrganizationIDs" type="[]string">
  Filter by organization membership. Prefix IDs with `+` (include) or `-` (exclude). Accepts up to 100 values.
</ParamField>

<ParamField path="CreatedAtBefore" type="*int64">
  Unix timestamp (milliseconds). Return users created before this time.
</ParamField>

<ParamField path="CreatedAtAfter" type="*int64">
  Unix timestamp (milliseconds). Return users created after this time.
</ParamField>

<ParamField path="LastActiveAtBefore" type="*int64">
  Unix timestamp (milliseconds). Return users last active before this time.
</ParamField>

<ParamField path="LastActiveAtAfter" type="*int64">
  Unix timestamp (milliseconds). Return users last active after this time.
</ParamField>

<ParamField path="LastSignInAtBefore" type="*int64">
  Unix timestamp (milliseconds). Return users who last signed in before this time.
</ParamField>

<ParamField path="LastSignInAtAfter" type="*int64">
  Unix timestamp (milliseconds). Return users who last signed in after this time.
</ParamField>

### Example

<CodeGroup>
```go Package-level
list, err := user.List(context.Background(), &user.ListParams{
    ListParams: clerk.ListParams{
        Limit:  clerk.Int64(20),
        Offset: clerk.Int64(0),
    },
    Query: clerk.String("jane"),
})
```

```go Client struct
list, err := c.List(context.Background(), &user.ListParams{
    ListParams: clerk.ListParams{
        Limit:  clerk.Int64(20),
        Offset: clerk.Int64(0),
    },
    Query: clerk.String("jane"),
})
```
</CodeGroup>
