---
title: 'Organizations'
description: 'Create and manage organizations in your Clerk instance.'
---

Import the `organization` package to manage organizations.

```go
import "github.com/clerk/clerk-sdk-go/v2/organization"
```

## Operations

### Create

Creates a new organization.

```go
func Create(ctx context.Context, params *CreateParams) (*clerk.Organization, error)
// Client method:
func (c *Client) Create(ctx context.Context, params *CreateParams) (*clerk.Organization, error)
```

<Tabs>
  <Tab title="Without client">
    ```go
    import (
        "context"
        "encoding/json"

        "github.com/clerk/clerk-sdk-go/v2"
        "github.com/clerk/clerk-sdk-go/v2/organization"
    )

    clerk.SetKey("sk_live_...")

    name := "Acme Inc"
    slug := "acme-inc"
    org, err := organization.Create(context.Background(), &organization.CreateParams{
        Name: &name,
        Slug: &slug,
    })
    ```
  </Tab>
  <Tab title="With client">
    ```go
    import (
        "context"

        "github.com/clerk/clerk-sdk-go/v2"
        "github.com/clerk/clerk-sdk-go/v2/organization"
    )

    client := organization.NewClient(&clerk.ClientConfig{})

    name := "Acme Inc"
    slug := "acme-inc"
    org, err := client.Create(context.Background(), &organization.CreateParams{
        Name: &name,
        Slug: &slug,
    })
    ```
  </Tab>
</Tabs>

#### CreateParams

<ParamField path="Name" type="*string">
  The name of the organization.
</ParamField>

<ParamField path="Slug" type="*string">
  The slug of the organization.
</ParamField>

<ParamField path="CreatedBy" type="*string">
  The ID of the user who created the organization.
</ParamField>

<ParamField path="MaxAllowedMemberships" type="*int64">
  The maximum number of memberships allowed in the organization.
</ParamField>

<ParamField path="RoleSetKey" type="*string">
  Preview field — not yet available for all customers. Using this field will cause an error to be returned.
</ParamField>

<ParamField path="PublicMetadata" type="*json.RawMessage">
  Arbitrary metadata to be stored alongside the organization, accessible to your frontend.
</ParamField>

<ParamField path="PrivateMetadata" type="*json.RawMessage">
  Arbitrary metadata to be stored alongside the organization, accessible only from the Backend API.
</ParamField>

#### ListParams

<ParamField path="Limit" type="int64">
  The maximum number of results to return. Inherited from `clerk.ListParams`.
</ParamField>

<ParamField path="Offset" type="int64">
  The number of results to skip. Inherited from `clerk.ListParams`.
</ParamField>

<ParamField path="IncludeMembersCount" type="*bool">
  When `true`, each organization in the response includes the `members_count` field.
</ParamField>

<ParamField path="IncludeMissingMemberWithElevatedPermissions" type="*bool">
  When `true`, each organization includes `missing_member_with_elevated_permissions`.
</ParamField>

<ParamField path="OrderBy" type="*string">
  The field to order results by. Prefix with `-` for descending order (e.g., `-created_at`).
</ParamField>

<ParamField path="Query" type="*string">
  Filters organizations whose name or slug match the query string.
</ParamField>

<ParamField path="UserIDs" type="[]string">
  Filters organizations that have at least one member with one of the given user IDs.
</ParamField>

<ParamField path="FilterBy" type="[]string">
  Additional filters to apply to the list.
</ParamField>

## Response types

### Organization

<ResponseField name="object" type="string">
  Always `"organization"`.
</ResponseField>

<ResponseField name="id" type="string">
  The unique identifier for the organization.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the organization.
</ResponseField>

<ResponseField name="slug" type="string">
  The slug of the organization, used in URLs.
</ResponseField>

<ResponseField name="image_url" type="string | null">
  The URL of the organization's logo image.
</ResponseField>

<ResponseField name="has_image" type="boolean">
  Whether the organization has a logo image.
</ResponseField>

<ResponseField name="members_count" type="integer">
  The total number of members. Only present when `include_members_count` is `true`.
</ResponseField>

<ResponseField name="missing_member_with_elevated_permissions" type="boolean">
  Only present when `include_missing_member_with_elevated_permissions` is `true`.
</ResponseField>

<ResponseField name="pending_invitations_count" type="integer">
  The number of pending invitations.
</ResponseField>

<ResponseField name="max_allowed_memberships" type="integer">
  The maximum number of memberships allowed in the organization.
</ResponseField>

<ResponseField name="membership_limit_set_by" type="string | null">
  Indicates who set the membership limit.
</ResponseField>

<ResponseField name="admin_delete_enabled" type="boolean">
  Whether organization admins can delete the organization via the Frontend API.
</ResponseField>

<ResponseField name="role_set_key" type="string | null">
  The key of the role set assigned to the organization. Preview field.
</ResponseField>

<ResponseField name="public_metadata" type="object">
  Arbitrary metadata accessible to your frontend.
</ResponseField>

<ResponseField name="private_metadata" type="object">
  Arbitrary metadata accessible only from the Backend API.
</ResponseField>

<ResponseField name="created_by" type="string">
  The ID of the user who created the organization.
</ResponseField>

<ResponseField name="created_at" type="integer">
  Unix timestamp (milliseconds) when the organization was created.
</ResponseField>

<ResponseField name="updated_at" type="integer">
  Unix timestamp (milliseconds) when the organization was last updated.
</ResponseField>

### OrganizationList

<ResponseField name="data" type="Organization[]">
  The list of organizations.
</ResponseField>

<ResponseField name="total_count" type="integer">
  The total number of organizations matching the query.
</ResponseField>
