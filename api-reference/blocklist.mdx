---
title: Blocklist Identifiers
description: Use the Clerk Go SDK to manage the blocklist, preventing specific email addresses or phone numbers from signing up.
---

The blocklist lets you prevent specific identifiers from completing sign-up. Any email address or phone number on the blocklist will be rejected during registration, even if the user otherwise meets all other criteria.

## Import

```go
import "github.com/clerk/clerk-sdk-go/v2/blocklistidentifier"
```

---

## Operations

### Create

Adds a new identifier to the blocklist.

```go
func Create(ctx context.Context, params *CreateParams) (*clerk.BlocklistIdentifier, error)
```

**Client method**

```go
func (c *Client) Create(ctx context.Context, params *CreateParams) (*clerk.BlocklistIdentifier, error)
```

#### CreateParams

<ParamField path="Identifier" type="*string">
  The email address or phone number to block (e.g. `"spammer@example.com"` or `"+15555550100"`).
</ParamField>

#### Example

<Tabs>
  <Tab title="Without client">
    ```go
    package main

    import (
      "context"

      "github.com/clerk/clerk-sdk-go/v2"
      "github.com/clerk/clerk-sdk-go/v2/blocklistidentifier"
    )

    func main() {
      clerk.SetKey("sk_live_...")

      identifier := "spammer@example.com"

      entry, err := blocklistidentifier.Create(context.Background(), &blocklistidentifier.CreateParams{
        Identifier: &identifier,
      })
      if err != nil {
        panic(err)
      }

      _ = entry
    }
    ```
  </Tab>

### Delete

Removes an identifier from the blocklist.

```go
func Delete(ctx context.Context, id string) (*clerk.DeletedResource, error)
```

**Client method**

```go
func (c *Client) Delete(ctx context.Context, id string) (*clerk.DeletedResource, error)
```

| Parameter | Type     | Description                                      |
|-----------|----------|--------------------------------------------------|
| `id`      | `string` | The ID of the blocklist identifier to remove.    |

### List

Returns all identifiers currently on the blocklist.

```go
func List(ctx context.Context, params *ListParams) (*clerk.BlocklistIdentifierList, error)
```

**Client method**

```go
func (c *Client) List(ctx context.Context, params *ListParams) (*clerk.BlocklistIdentifierList, error)
```

#### ListParams

`ListParams` has no fields beyond the embedded `clerk.APIParams`. Pass an empty struct to retrieve all entries.

#### Example

<Tabs>
  <Tab title="Without client">
    ```go
    package main

    import (
      "context"
      "fmt"

      "github.com/clerk/clerk-sdk-go/v2"
      "github.com/clerk/clerk-sdk-go/v2/blocklistidentifier"
    )

    func main() {
      clerk.SetKey("sk_live_...")

      list, err := blocklistidentifier.List(context.Background(), &blocklistidentifier.ListParams{})
      if err != nil {
        panic(err)
      }

      fmt.Printf("Total: %d\n", list.TotalCount)
      for _, entry := range list.BlocklistIdentifiers {
        fmt.Println(entry.Identifier, entry.IdentifierType)
      }
    }
    ```
  </Tab>
  <Tab title="With client">

## Response types

### BlocklistIdentifier

Returned by `Create` and as elements in `List`.

<ResponseField name="object" type="string">
  Always `"blocklist_identifier"`.
</ResponseField>

<ResponseField name="id" type="string">
  Unique identifier for this blocklist entry.
</ResponseField>

<ResponseField name="identifier" type="string">
  The email address or phone number that is blocked.
</ResponseField>

<ResponseField name="identifier_type" type="string">
  The type of identifier. Typically `"email_address"` or `"phone_number"`.
</ResponseField>

<ResponseField name="created_at" type="int64">
  Unix timestamp (milliseconds) when the entry was created.
</ResponseField>

<ResponseField name="updated_at" type="int64">
  Unix timestamp (milliseconds) when the entry was last updated.
</ResponseField>

### BlocklistIdentifierList

Returned by `List`.

<ResponseField name="data" type="[]*BlocklistIdentifier">
  All identifiers currently on the blocklist.
</ResponseField>

<ResponseField name="total_count" type="int64">
  Total number of entries in the blocklist.
</ResponseField>

### DeletedResource

Returned by `Delete`.

<ResponseField name="id" type="string">
  The ID of the deleted blocklist identifier.
</ResponseField>

<ResponseField name="object" type="string">
  The object type that was deleted.
</ResponseField>

<ResponseField name="deleted" type="boolean">
  `true` when the resource was successfully deleted.
</ResponseField>
