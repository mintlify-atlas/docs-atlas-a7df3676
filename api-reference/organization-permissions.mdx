---
title: 'Organization permissions'
description: 'Create, retrieve, update, delete, and list custom organization permissions.'
---

Import the `organizationpermission` package to manage organization permissions.

```go
import "github.com/clerk/clerk-sdk-go/v2/organizationpermission"
```

## Operations

### Create

Creates a new organization permission.

```go
func Create(ctx context.Context, params *CreateParams) (*clerk.OrganizationPermission, error)
// Client method:
func (c *Client) Create(ctx context.Context, params *CreateParams) (*clerk.OrganizationPermission, error)
```

<Tabs>
  <Tab title="Without client">
    ```go
    import (
        "context"

        "github.com/clerk/clerk-sdk-go/v2"
        "github.com/clerk/clerk-sdk-go/v2/organizationpermission"
    )

    clerk.SetKey("sk_live_...")

    name := "Read Invoices"
    key := "org:billing:read"
    description := "Allows reading billing invoices"
    permission, err := organizationpermission.Create(context.Background(), &organizationpermission.CreateParams{
        Name:        &name,
        Key:         &key,
        Description: &description,
    })
    ```
  </Tab>
  <Tab title="With client">
    ```go
    import (
        "context"

        "github.com/clerk/clerk-sdk-go/v2"
        "github.com/clerk/clerk-sdk-go/v2/organizationpermission"
    )

    client := organizationpermission.NewClient(&clerk.ClientConfig{})

    name := "Read Invoices"
    key := "org:billing:read"
    description := "Allows reading billing invoices"
    permission, err := client.Create(context.Background(), &organizationpermission.CreateParams{
        Name:        &name,
        Key:         &key,
        Description: &description,
    })
    ```
  </Tab>
</Tabs>

#### CreateParams

<ParamField path="Name" type="*string">
  The display name of the permission.
</ParamField>

<ParamField path="Key" type="*string">
  The unique key used to identify the permission in code (e.g., `"org:billing:read"`).
</ParamField>

<ParamField path="Description" type="*string">
  A human-readable description of the permission.
</ParamField>

### List

```go
func List(ctx context.Context, params *ListParams) (*clerk.OrganizationPermissionList, error)
// Client method:
func (c *Client) List(ctx context.Context, params *ListParams) (*clerk.OrganizationPermissionList, error)
```

<Tabs>
  <Tab title="Without client">

    ```go
        "context"
    ```

  </Tab>
</Tabs>

#### ListParams

<ParamField path="Limit" type="int64">
  Maximum number of results to return. Inherited from `clerk.ListParams`.
</ParamField>

<ParamField path="Offset" type="int64">
  Number of results to skip. Inherited from `clerk.ListParams`.
</ParamField>

<ParamField path="Query" type="*string">
  Filters permissions whose name or key matches the search query.
</ParamField>

<ParamField path="OrderBy" type="*string">
  The field to order results by. Prefix with `-` for descending order.
</ParamField>

## Response types

### OrganizationPermission

<ResponseField name="object" type="string">
  Always `"permission"`.
</ResponseField>

<ResponseField name="id" type="string">
  The unique identifier for the permission.
</ResponseField>

<ResponseField name="name" type="string">
  The display name of the permission.
</ResponseField>

<ResponseField name="key" type="string">
  The unique key used to identify the permission in code.
</ResponseField>

<ResponseField name="description" type="string | null">
  A human-readable description of the permission.
</ResponseField>

<ResponseField name="type" type="string">
  The type of the permission.
</ResponseField>

<ResponseField name="created_at" type="integer">
  Unix timestamp (milliseconds) when the permission was created.
</ResponseField>

<ResponseField name="updated_at" type="integer">
  Unix timestamp (milliseconds) when the permission was last updated.
</ResponseField>

### OrganizationPermissionList

<ResponseField name="data" type="OrganizationPermission[]">
  The list of organization permissions.
</ResponseField>

<ResponseField name="total_count" type="integer">
  The total number of permissions matching the query.
</ResponseField>
