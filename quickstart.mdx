<Steps>

<Step title="Install the SDK">

Add the SDK to your Go module:

```bash
go get -u github.com/clerk/clerk-sdk-go/v2
```

Or, if you use Go modules and prefer to let the toolchain resolve the dependency on the next build, add the import directly to your Go file — the module will be fetched automatically when you run `go mod tidy`.

```go
import "github.com/clerk/clerk-sdk-go/v2"
```

</Step>

<Step title="Set your secret key">

Find your secret key in the [Clerk Dashboard](https://dashboard.clerk.com) under **API Keys**. Call `clerk.SetKey` once at application startup.

```go
package main

import "github.com/clerk/clerk-sdk-go/v2"

func main() {
    clerk.SetKey("sk_live_XXX")
}
```

<Warning>
Never hard-code your secret key in source code. Read it from an environment variable (for example, `os.Getenv("CLERK_SECRET_KEY")`) before passing it to `clerk.SetKey`.
</Warning>

</Step>

<Step title="Make your first API call">

Import the resource package for what you need and call the operation. Every operation takes a `context.Context` as its first argument.

<CodeGroup>

```go list users
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/clerk/clerk-sdk-go/v2"
    "github.com/clerk/clerk-sdk-go/v2/user"
)

func main() {
    clerk.SetKey("sk_live_XXX")

    users, err := user.List(context.Background(), &user.ListParams{})
    if err != nil {
        log.Fatal(err)
    }

    fmt.Printf("Total users: %d\n", users.TotalCount)
    for _, u := range users.Users {
        fmt.Println(u.ID)
    }
}
```

```go create organization
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/clerk/clerk-sdk-go/v2"
    "github.com/clerk/clerk-sdk-go/v2/organization"
)

func main() {
    clerk.SetKey("sk_live_XXX")

    org, err := organization.Create(context.Background(), &organization.CreateParams{
        Name: clerk.String("Clerk Inc"),
    })
    if err != nil {
        log.Fatal(err)
    }

    fmt.Printf("Created organization: %s\n", org.ID)
}
```

</CodeGroup>

</Step>

<Step title="Add HTTP middleware (optional)">

Protect your HTTP handlers with Clerk's built-in middleware. `WithHeaderAuthorization` verifies the `Authorization: Bearer <token>` header and writes the session claims into the request context. `RequireHeaderAuthorization` does the same but returns HTTP 403 if no valid claims are found.

```go
package main

import (
    "fmt"
    "net/http"

    "github.com/clerk/clerk-sdk-go/v2"
    clerkhttp "github.com/clerk/clerk-sdk-go/v2/http"
)

func main() {
    clerk.SetKey("sk_live_XXX")

    mux := http.NewServeMux()
    mux.HandleFunc("/", publicRoute)

    protectedHandler := http.HandlerFunc(protectedRoute)
    mux.Handle(
        "/protected",
        clerkhttp.WithHeaderAuthorization()(protectedHandler),
    )

    http.ListenAndServe(":3000", mux)
}

func publicRoute(w http.ResponseWriter, r *http.Request) {
    w.Write([]byte(`{"access": "public"}`))
}

func protectedRoute(w http.ResponseWriter, r *http.Request) {
    claims, ok := clerk.SessionClaimsFromContext(r.Context())
    if !ok {
        w.WriteHeader(http.StatusUnauthorized)
        w.Write([]byte(`{"access": "unauthorized"}`))
        return
    }
    fmt.Fprintf(w, `{"user_id": "%s"}`, claims.Subject)
}
```

<Tip>
Use `clerkhttp.RequireHeaderAuthorization()` instead of `WithHeaderAuthorization` if you want the middleware to automatically respond with HTTP 403 when no valid session token is present, rather than passing the request through unauthenticated.
</Tip>

</Step>
